# SHRIMPSAY

1. SETUP
```
chmod +x shrimpsay
```
2. RUN
```
./shrimpsay -s "Bom dia"
```
## OPTIONS

-s "insert speak"

-a (sword item)

-b (mud item)

-c (speaker)

-d (UZI)

-e "insert your own item"
