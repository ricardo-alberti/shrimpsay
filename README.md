# SHRIMPSAY

![shrimpsay output](/shrimpsay.png "shrimpsay output")

1. SETUP
```
chmod +x shrimpsay
```
2. RUN
```
./shrimpsay -s "Bom dia"
```
## OPTIONS

-s "insert speak"

-a (sword item)

-b (mug item)

-c (speaker)

-d (UZI)

-e "insert your own item"

### Example: 
```
./shrimpsay -s "Bom dia" -b 
```
